# SnippetCentre

### Descripción
Proyecto personal para practicar .Net antes de rendir mi final de la materia.
La aplicación trata de un simple almacen de fragmentos de codigo, podrás crear categorías de codigo/lenguajes donde dentro de ellos se almacenaran cualquier cantidad de snippets.
Las categorías pueden ser creadas, editadas y eliminadas sin problema, los snippets pertenecientes a estas podrán hacer lo mismo. 
En el proyecto principal el almacenamiento de estos se produce en un archivo, en otra rama localmente en un sql server, y en la tercera para repasar se integrará con una api y algun proyecto blazor.

### Tecnologías 

- .Net 10
- Windows Forms
- ASP.NET api
- Entity framework
- Blazor
- ADO.net
